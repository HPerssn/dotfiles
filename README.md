# My Dotfiles

Personal configuration files for tmux and neovim.

## Installation
```bash
git clone
cd ~/dotfiles
./install.sh
```

## Contents

- `nvim/` - Neovim configuration
- `tmux/` - Tmux configuration

## Requirements

- Neovim >= 0.8
- Tmux >= 3.0
- Git

## Notes

The install script will backup any existing configs to `.backup` files.
